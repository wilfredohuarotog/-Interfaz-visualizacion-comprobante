## Ejecución

### 1. Clonar el repositorio
```bash
git clone https://github.com/wilfredohuarotog/-Interfaz-visualizacion-comprobante.git
```
### 2. Ingresar al directorio
```
cd -Interfaz-visualizacion-comprobante
```
### 3. Instalar dependencias
```bash
npm install
```
### 4. Levantar la app
```bash
npm start
```
